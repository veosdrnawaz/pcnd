🚀 Excited to share my latest project: PakNotes AI — Instant Pakistani Currency Note Distribution!

During my learning journey, I wanted to build a practical tool that sits at the intersection of Machine Learning, Finance, and Modern Web Development. 

💵 The Challenge:
ATM operations, bank cash desks, and retailers constantly need to distribute cash denominations efficiently. How can we automate this layout while guaranteeing change rules (such as ensuring at least one Rs.10 note is always included for cash breakdowns)?

💡 The Solution:
PakNotes AI uses a hybrid pipeline to predict the optimal breakdown of Rs.1000, Rs.500, Rs.100, Rs.50, Rs.20, and Rs.10 notes for any input amount up to Rs. 500,000.

🛠️ The Tech Stack:
- Backend: FastAPI (Python) for asynchronous, high-speed API endpoints.
- Machine Learning: Trained a Scikit-Learn RandomForestRegressor (wrapped in a MultiOutputRegressor) on a custom dataset of 50,000 generated distribution logs.
- Math Verification: A validation layer checks model outputs and falls back to a deterministic algorithm if there's any rounding mismatch, guaranteeing 100% mathematical accuracy.
- Frontend: A premium glassmorphism dashboard built with responsive HTML5, Vanilla CSS, and JS. It supports Dark/Light theme toggles, local storage logs, and client-side exports to CSV and PDF!
